# Kubernetes Hello World Test API

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript-based API.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# unit tests with test coverage
$ yarn test:coverage
```
